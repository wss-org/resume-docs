# resume-docs

## 安装

```shell
npm install @serverless-devs/s@2.x -g  
```

配置[阿里云密钥](https://docs.serverless-devs.com/fc/config)


## 初始化并启动项目

```
s init resume

cd resume

npm install
# yarn install

npm run start
```

## 修改项目

修改 src/index.md 文件，刷新页面
